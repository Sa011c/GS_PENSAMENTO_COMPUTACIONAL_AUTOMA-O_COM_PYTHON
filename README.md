## 📋 Sobre o Projeto

O **Synapse Mission Control AI** é um sistema desenvolvido para a disciplina de **Pensamento Computacional e Automação com Python**.

O projeto simula o monitoramento de uma missão espacial, realizando análises automáticas dos dados coletados e auxiliando na tomada de decisões por meio de Inteligência Artificial.

---

## 🎯 Objetivos do Sistema

O sistema é capaz de:

* Armazenar dados simulados de uma missão espacial;
* Analisar diferentes ciclos de monitoramento;
* Gerar alertas automáticos;
* Calcular o nível de risco de cada ciclo;
* Identificar a tendência da missão;
* Indicar qual área da missão foi mais afetada;
* Exibir um relatório final com os resultados obtidos.

---

## 🛠️ Tecnologias Utilizadas

* Python

---

## ⚙️ Funcionalidades

* Monitoramento de temperatura, energia e comunicação;
* Classificação automática de status (Normal, Atenção e Crítico);
* Geração de alertas em tempo real;
* Simulação automática de ciclos;
* Inserção manual de dados para testes;
* Diagnóstico inteligente utilizando IA;
* Histórico dos diagnósticos realizados.
